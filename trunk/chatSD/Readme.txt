ATENCIÓN: LEA ESTE ARCHIVO SI NO TIENE UN SERVIDOR AL CUAL CONECTARSE

El presente programa es un programa Cliente de mensajería, por lo que requiere
de un servidor al cual conectarse.

Si desea saber como instalar un servidor, en el directorio de instalación de 
este programase encuentra una carpeta llamada "Servidor", donde encontrará los 
archivos necesarios para montar un servidor.
Para obtener más detalles e instrucciones lea el archivo "Manual_Tecnico.pdf" 
que se encuentra en la carpeta "Servidor",dentro del directorio de instalación del 
programa.
